# 관상 + 사주팔자 AI 앱 기획서

> 작성일: 2026-06-12

---

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 천명관상 (天命觀相) *(가칭)* |
| 한 줄 소개 | 사주팔자와 관상을 AI로 결합한 나만의 운명 풀이 |
| 플랫폼 | PWA (iOS/Android 홈화면 설치) |
| 배포 | GitHub Pages |
| 핵심 가치 | 타고난 운명(사주) + 얼굴에 새겨진 운(관상)을 함께 읽어 더 정확한 풀이 제공 |

---

## 2. 사용자 플로우

```
[앱 진입]
    ↓
[1단계] 성별 선택
    남 / 여
    ↓
[2단계] 생년월일시 입력
    년(4자리) / 월 / 일 / 시(12지 시각)
    → 사주팔자 8글자 계산
    ↓
[3단계] 얼굴 사진 촬영 or 갤러리 업로드
    → AI Vision으로 관상 분석
    ↓
[4단계] 결과 화면
    → 등급 발표 + 종합 풀이 텍스트
    → 공유하기 버튼
```

---

## 3. 핵심 기능 상세

### 3-1. 사주팔자 계산
- 입력: 성별, 생년(년), 생월(월), 생일(일), 생시(시)
- 출력: 천간(天干) 4자 + 지지(地支) 4자 = 사주 8글자
- 계산 방식: 만세력 기반 JavaScript 로직 (API 불필요, 오프라인 가능)
- 오행(五行) 분석: 목·화·토·금·수 비율 도출
- 일간(日干) 기준 격국(格局) 판단

### 3-2. 관상 AI 분석
- 입력: 얼굴 사진 (카메라 or 갤러리)
- API: OpenRouter → Google Gemini Flash (Vision, 무료)
- 분석 항목:
  - 이마 (초년운, 지성)
  - 눈 (중년운, 통찰력)
  - 코 (재물운)
  - 입/턱 (말년운, 의지력)
  - 귀 (선천운)
  - 전체적인 인상 (기운, 오행 매칭)

### 3-3. 통합 풀이 생성
- 사주 분석 결과 + 관상 분석 결과를 합쳐 AI 프롬프트로 전달
- 최종 등급 판정 + 종합 서사형 풀이 텍스트 생성
- 프롬프트 예시:
  > "이 사람의 사주는 [일간: 甲木, 오행 구성: 목30% 화20% 토10% 금30% 수10%]이며,
  > 관상 분석 결과는 [이마 넓음, 눈이 깊고 광채 있음, 코 높고 살집 있음]입니다.
  > 성별은 여성입니다. 사주와 관상을 통합하여 등급을 판정하고 풀이를 작성해 주세요."

## 7. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| Frontend | HTML/CSS/JS | 기존 앱과 동일, 빠른 개발 |
| AI Vision | OpenRouter (Gemini Flash Free) | 칼로리앱에서 검증됨 |
| AI 풀이 생성 | OpenRouter (Gemini Flash Free) | 동일 API로 통합 |
| 사주 계산 | 순수 JS 로직 | API 불필요, 오프라인 작동 |
| 저장소 | 없음 (서버리스) | 개인정보 보호, 단순 구조 |
| 배포 | GitHub Pages | 무료, 즉시 배포 |
| PWA | manifest.json + sw.js | 아이폰 홈화면 설치 |

## 8. 디자인 방향

- **컨셉**: 동양 신비주의 + 모던 UI
- **컬러**: 진한 남색(#1a1a3e) 배경 + 금색(#c9a84c) 포인트
- **폰트**: 나눔명조 or Noto Serif KR (한자 분위기)
- **아이콘**: 별, 왕관, 팔괘(☯), 오행 심볼
- **애니메이션**: 결과 등급 등장 시 극적인 reveal 효과

---

## 9. 개발 순서

1. 프로젝트 초기 세팅 (HTML/CSS/JS 기본 구조)
2. 사주팔자 계산 JS 로직 구현
3. 얼굴 사진 촬영/업로드 기능
4. OpenRouter Vision API 연동 (관상 분석)
5. 통합 풀이 프롬프트 설계 및 결과 생성
6. 등급 시스템 + 결과 화면 UI
7. PWA 설정 (manifest, sw.js)
8. GitHub Pages 배포
